нужно отразить по горизонтали картинки
front  и right.
Картинку top нужно отразить по горизонтали
и повернуть на 180 градусов.

Анимированные спрайты добавляются в виде спрайтшитов.
Высота спрайтшита - 1 фрейм анимации.
Ширина - количество фреймов анимации х ширина фрейма.
Ширина фрейма берётся из ширины патча.

Параметр depth в патчах используется для сортировки
в порядке отрисовки патчей.

В пайплайн отрисовки панорам можно добавить масштабирование
перед отрисовкой в спрайт. Примеры вы локациях "Лаборатория"
И "Клетка".

Пример хотспотов в локации "Клетка".
