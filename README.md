Анимированные спрайты добавляются в виде спрайтшитов.
Высота спрайтшита - 1 фрейм анимации.
Ширина - количество фреймов анимации х ширина фрейма.
Ширина фрейма берётся из ширины патча.

Параметр depth в патчах используется для сортировки
в порядке отрисовки патчей. Параметр "active" указывает,
будет ли патч отображаться.

В пайплайн отрисовки панорам можно добавить масштабирование
перед отрисовкой в спрайт. Примеры в локациях "Лаборатория"
И "Клетка".

Пример хотспотов в локации "Клетка". У хотспотов есть параметр
"cursor", который определяет вид курсора при наведении на хотспот.
Параметр "active" указывает, будет ли хотспот реагировать на клик
или наведение курсора.

Добавлен параметр "action" для патчей и хотспотов.
Он предназначен для функций реакции на клик в случае хотспотов И
И функций, выполняющихся по таймеру, в случае патчей (при условии, что
параметр патча "is_active" равен true.
